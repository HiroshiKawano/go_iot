- [x] 2. 集計帳票（日次/時間別の Go バケット集計と描画）
- [x] 2.1 日次/時間別バケット集計ロジック
  - 期間内全行を JST 暦日／JST 時間帯（hour-of-day 0-23）でバケット化し、温度・湿度それぞれの 平均/最高/最低/日較差/標準偏差/CV と、各バケットの適正帯滞在率を組み立てる（純粋層 `internal/chart` を流用し time は handler 境界で扱う）。
  - 適正帯滞在率はデバイスの作物の VPD 適正帯（未設定/不正は既定 0.3〜1.5 kPa）に対する在帯割合（瞬間値/平均ではなくバケット期間の在帯割合）として算出する。
  - テーブル駆動テスト: JST 日跨ぎ・空バケット/欠測日は「—」・単点で σ/CV 未定義は「—」・空期間は空帳票・作物設定あり/なしで適正帯が切り替わること。
  - 観測可能な完了: 日次/時間別バケットの単体テストが緑になり、欠測「—」と作物別適正帯滞在率が算出される。
  - _Requirements: 4.1, 4.2, 4.3, 4.4, 5.1, 5.2, 5.3, 6.1, 6.2, 6.3, 6.4, 6.5_
  - _Boundary: readings_report builders_
  - _Depends: 1.1_
- [x] 2.2 帳票・CSV ボタンの View 構造とフラグメント描画

  - 結果領域フラグメントへ、日次/時間別の `.data-table` と CSV ダウンロードボタン（HTMX 非対象のファイルDL リンク）をモック写経で描画する。CSV リンク・帳票の値はハンドラが渡す（4.1）。
  - templ レンダリングテスト: 手組みの View から帳票表（`.data-table`）・作物/適正帯ラベル・CSV ボタン（href にリンク文字列）が描画され、空期間では帳票が安全に空表示になる。
  - 観測可能な完了: フラグメントを Render した HTML に日次/時間別の帳票表・作物/適正帯ラベル・CSV ボタンが含まれる。
  - _Requirements: 4.1, 4.4, 5.1, 5.3, 6.1_

- [x] 3. CSV エクスポート（整形ヘルパとファイル応答ハンドラ）
- [x] 3.1 CSV 整形ヘルパとファイル名生成
  - 期間内の全計測行を CSV へ書き出すヘルパを実装する（UTF-8 BOM 付与・日本語ヘッダ・各行にメタ列〔デバイスID/デバイス名/地点/作物〕を反復付与・選択項目で温度/湿度の列を増減・項目未選択は両方を既定・計測日時は JST RFC3339・エスケープは標準 CSV に委任）。地点/作物は表示名解決し未設定は「未設定」。
  - 添付ファイル名を ASCII フォールバック＋RFC 5987 `filename*`（日本語デバイス名）で生成するヘルパを実装する。
  - 単体テスト: 先頭 BOM 3バイト・日本語ヘッダ・メタ列の行反復・項目フィルタの列増減・カンマ/改行/引用符を含むデバイス名のエスケープ（`csv` パーサ往復）・空行でヘッダのみ・日本語ファイル名のエンコード。
  - 観測可能な完了: ヘルパの単体テストが緑になり、出力バイト列が BOM＋日本語ヘッダ＋メタ列付きデータ行になる。
  - _Requirements: 1.1, 1.2, 1.3, 1.4, 1.5, 2.1, 2.2, 2.3, 3.1, 3.2, 3.4_
  - _Boundary: readings_export CSV helpers_
- [x] 3.2 CSV エクスポートハンドラ（認可・形式不正・ストリーミング応答）
  - CSV ダウンロード経路のハンドラを実装する: 所有者認可（非所有/不在→404 列挙防止）→ 期間フィルタを既存写像で確定（形式不正→400・データ無出力）→ 全行取得 → `text/csv; charset=utf-8` ＋ `Content-Disposition: attachment` ＋ファイル名で逐次書き出し（`Flush` で送出）する。期間フィルタは画面一覧と同一区間を共有する。
  - httptest: 正常（200・ヘッダ・BOM・全行）・項目フィルタ（温度のみで湿度列なし）・空期間（ヘッダのみ）・形式不正（400）・非所有（404＋DB 書込副作用なし）・DB エラー（500）・大量行で全行出力。
  - 観測可能な完了: `GET /devices/:device/readings.csv` が添付 CSV を返し、非所有→404・形式不正→400・大量行でも完了する。
  - _Requirements: 1.1, 1.6, 1.7, 2.1, 3.3, 7.1, 7.3, 8.1, 8.2, 9.1, 9.2_
  - _Boundary: ReadingsHandler.Export_
  - _Depends: 1.1, 3.1_
